Testing in Quality Control

Testing methods, procedures, and documentation including FAT, SAT, and commissioning tests.

What is Testing?

Testing is the process of verifying that a system, component, or product meets its specified requirements and functions correctly under defined conditions. It is a critical part of quality control to ensure reliability, safety, and performance.

Types of Testing

  • Factory Acceptance Test (FAT) – At manufacturer's facility
  • Site Acceptance Test (SAT) – At customer's facility
  • Commissioning – System startup and functional verification
  • Performance Testing – Under load conditions
  • Environmental Testing – Temperature, humidity, vibration
  • Electrical Testing – Continuity, insulation, hi-pot, loop checks

Testing Documentation

  • Test Procedure – Step-by-step instructions
  • Test Plan – Schedule and resource allocation
  • Test Report – Results and sign-off
  • Non-Conformance Report (NCR) – For defects found

FAT Procedure Example

# FAT Procedure for Control Panel # 1. Visual Inspection – Verify panel construction # 2. Continuity Testing – Check all wiring connections # 3. Insulation Resistance Test – > 1 MΩ # 4. Power-up Test – Verify power supply voltages # 5. Function Test – Test all I/O and control functions # 6. Simulation Test – Simulate field signals # 7. Communication Test – Verify network connectivity # 8. Documentation – Submit test report

Example: FAT for PLC Panel

A FAT for a PLC control panel involves testing all input/output modules, verifying communication with HMI and SCADA, simulating process inputs, and checking alarm and interlock functions. A signed test report is required before shipping to site.
