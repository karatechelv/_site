EasyBuilder

HMI design software for Weintek and compatible panels – intuitive and powerful.

Overview

EasyBuilder Pro is the HMI design software for Weintek HMIs and compatible panels. It provides an intuitive drag-and-drop interface for creating operator interfaces, with support for various communication protocols and advanced features.

Key Features

  • Drag-and-Drop Design – Easy and intuitive
  • Multi-Protocol Support – Modbus, Profinet, Ethernet/IP
  • Macros and Scripting – Advanced logic and automation
  • Remote Monitoring – Web server and VNC
  • Recipe Management – Data storage and recall

Learning Path

  • EasyBuilder Basics – Installation and interface
  • Creating Screens – Objects and graphics
  • Tag Management – Connecting to PLC
  • Macros and Scripts – Advanced functionality
  • Remote Access – Web server and VNC setup

Example: HMI for Pump Control

EasyBuilder Pro is used to create an HMI screen for pump control with start/stop buttons, status indicators, pressure gauges, and alarm logs. The HMI communicates with a PLC via Modbus RTU for real-time data exchange.
