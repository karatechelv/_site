Factory IO

3D simulation software for PLC training and testing with virtual factories and industrial scenes.

Overview

Factory IO is a 3D simulation software designed for PLC training and testing. It provides virtual industrial scenes with sensors, actuators, and conveyors that respond to PLC logic, making it ideal for learning and testing automation programs.

Key Features

  • 3D Virtual Scenes – Realistic industrial environments
  • PLC Connectivity – Siemens, Rockwell, Codesys
  • Built-in Scenes – Conveyor, sorting, pick-and-place
  • Customizable – Create your own scenes
  • Educational – Ideal for training and testing

Learning Path

  • Factory IO Basics – Interface and setup
  • Connecting PLC – TIA Portal, RSLogix, Codesys
  • Built-in Scenes – Conveyor, sorting, robotic arm
  • Creating Custom Scenes – Design your own
  • Advanced – HMI integration and complex logic

Example: Conveyor Sorting System

Factory IO simulates a conveyor sorting system with a PLC controlling the belt speed and sorting mechanisms. Students can write the PLC program and test it in the virtual environment before deploying to hardware.
