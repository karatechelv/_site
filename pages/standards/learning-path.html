Standards Learning Path

A structured learning path from foundation to advanced topics in engineering standards.

Foundation Level

  • Introduction to Engineering Standards
  • Overview of Standardization Organizations
  • Understanding Standards Terminology
  • Introduction to IEC, ISO, IEEE

Intermediate Level

  • IEC 61439 – Switchgear Standards
  • NEC (NFPA 70) – Electrical Code
  • ISO 9001 – Quality Management Systems
  • IEEE 80 – Earthing & Grounding
  • API RP 551 – Process Instrumentation

Advanced Level

  • Functional Safety Standards (IEC 61508/61511)
  • ISO 14001 & 45001 – Environment & Safety
  • IEEE 1584 – Arc Flash Hazard Analysis
  • API Q1 – Quality Management for Oil & Gas
  • Standards Auditing & Compliance

Suggested Path Flow

Foundation → IEC Overview → NEC & ISO 9001 → IEEE 80 & API RP 551 → Functional Safety → Advanced Standards

Each step connects to previous, related, and next topics for fluid learning.
