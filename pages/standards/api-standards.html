API Standards

American Petroleum Institute standards for oil and gas industry equipment, instrumentation, and safety.

What is API?

The American Petroleum Institute (API) is the leading trade association for the oil and natural gas industry. It develops standards for equipment, safety, and environmental protection in upstream and downstream operations.

Key API Standards

  • API RP 551 – Process Measurement Instrumentation
  • API RP 552 – Calibration of Transmitters
  • API RP 553 – Refinery Control Valves
  • API RP 554 – Process Instrumentation and Control
  • API RP 555 – Analyzers for Process Control
  • API RP 556 – Safety Instrumented Systems

API Q1 – Quality Management

  • API Q1 – Quality Management System Requirements
  • API Q2 – Quality Management for Service Suppliers
  • API Monogram Program – Product Certification
  • API 21 CFR – Regulatory Compliance

Example: API RP 551 – Instrumentation

API RP 551 provides guidelines for process measurement instrumentation in refineries and petrochemical plants. It covers selection, installation, calibration, and maintenance of instruments. Ensuring compliance with API standards is critical for safety and reliability in oil and gas facilities.
