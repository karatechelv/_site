Factory Acceptance Test (FAT)

Testing and verifying the panel at the manufacturer's facility before shipment.

What is FAT?

Factory Acceptance Test (FAT) is a comprehensive testing procedure conducted at the manufacturer's facility to verify that the panel meets all specifications and functions correctly before shipment. It ensures that the panel is ready for installation and reduces risks at site.

FAT Procedure Steps

  • Visual Inspection – Panel construction and labeling
  • Continuity Testing – Wiring verification
  • Insulation Test – Megger test for insulation
  • Power Up – Supply voltage verification
  • Function Test – I/O and control logic testing
  • Communication Test – Networking verification
  • Safety Check – Grounding and protection
  • Documentation – Test results and sign-off

FAT Documentation

  • Test Procedure – Step-by-step test plan
  • Test Results – Pass/fail records
  • Non-Conformance Report – For any issues found
  • Sign-off Document – Client approval
  • As-built Drawings – Updated if changes were made

Example: FAT for MCC Panel

A FAT for an MCC panel tests all motor starters, verifies PLC logic, checks communication with SCADA, and simulates field signals. The client observes the test and signs off if all requirements are met. Any issues are documented and corrected before shipment.
