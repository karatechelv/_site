Loop Check Phase

Verifying instrument loops and control circuits after installation.

What is Loop Check?

Loop check is the process of verifying each instrument loop from the field device to the control system. It ensures that sensors, transmitters, controllers, and actuators are correctly wired, configured, and communicating properly.

Loop Check Procedure

  • Identify Loop – Verify tag numbers and loop diagrams
  • Visual Inspection – Check wiring and terminations
  • Continuity Test – Verify signal path
  • Inject Signal – Simulate input from field device
  • Verify Indication – Check reading at PLC/DCS
  • Verify Output – Test actuator response
  • Document Results – Record loop check sheet

Loop Types

  • Analog Input Loop – 4-20mA from transmitter to PLC/DCS
  • Analog Output Loop – 4-20mA from PLC/DCS to valve
  • Digital Input Loop – Switch/sensor to PLC/DCS
  • Digital Output Loop – PLC/DCS to relay/solenoid
  • Communication Loop – HART, Fieldbus, Profibus

Example: Instrument Loop Verification

An instrument loop verification for a pressure transmitter includes: applying pressure to the transmitter, reading the 4-20mA signal at the PLC, and verifying the displayed value matches the applied pressure. The loop check sheet is signed off by the engineer.
