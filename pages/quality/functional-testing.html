✅ Functional Testing

Verifying system functionality, performance, and behavior under normal and abnormal operating conditions.

🔹 What is Functional Testing?

Functional testing is the process of verifying that a system or component performs its intended functions correctly under specified conditions. It involves testing all features, inputs, outputs, and responses to ensure they meet requirements.

🔹 Types of Functional Tests

  • Unit Testing – Testing individual components
  • Integration Testing – Testing combined components
  • System Testing – Testing the complete system
  • Regression Testing – Testing after changes
  • User Acceptance Testing (UAT) – Testing by end users

🔹 Functional Testing Process

  • Test Planning – Define scope and test cases
  • Test Execution – Run tests and record results
  • Defect Reporting – Log and track issues
  • Retesting – Verify fixes
  • Sign-off – Final approval
