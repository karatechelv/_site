🛠️ 7 QC Tools

The seven basic quality control tools for problem solving and process improvement.

🔹 What are the 7 QC Tools?

The 7 QC tools are a set of graphical and statistical techniques used for quality control and problem solving. They were developed by Kaoru Ishikawa and are widely used in manufacturing and service industries to identify, analyze, and resolve quality issues.

🔹 The 7 Tools

  • Cause and Effect Diagram (Fishbone/Ishikawa) – Root cause analysis
  • Check Sheet – Data collection and tallying
  • Control Chart – Monitoring process stability
  • Histogram – Frequency distribution
  • Pareto Chart – Prioritizing problems (80/20 rule)
  • Scatter Diagram – Correlation analysis
  • Stratification – Separating data into groups

🔹 Applications

  • Identifying root causes of defects
  • Monitoring process performance
  • Prioritizing improvement efforts
  • Analyzing data and trends
  • Problem solving and decision making
