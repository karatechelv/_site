Voltage Drop Calculator

📖 Description

Calculate voltage drop in electrical circuits based on cable length, current, and conductor size. Essential for proper cable sizing and system design.

📌 Example

For a 100m cable carrying 50A with 25mm² copper conductor, voltage drop = 5.2V (2.3% of 230V).
