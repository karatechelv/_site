Instrument Loop Diagrams

Instrument loop diagrams (Hook-up diagrams) for designing and troubleshooting instrument loops.

What are Loop Diagrams?

Loop diagrams (also called Hook-up diagrams) are detailed drawings that show the complete wiring and tubing connections for an instrument loop. They include all field devices, junction boxes, cables, and terminations in the loop.

Loop Diagram Elements

  • Instrument Tag Number – Unique identifier (e.g., PT-101)
  • Field Device – Sensor, transmitter, or valve
  • Wiring Details – Cable type, size, color code
  • Tubing Details – Material, size, connection type
  • Junction Boxes and Terminals
  • DCS/PLC Input/Output Channels
  • Power Supply and Grounding

Loop Types

  • Analog Input Loop – 4-20mA from transmitter to PLC/DCS
  • Analog Output Loop – 4-20mA from PLC/DCS to valve
  • Discrete Input Loop – Digital signal from switch/sensor
  • Discrete Output Loop – Digital output to relay/solenoid
  • HART Digital Loop – Digital communication over 4-20mA

Loop Diagram Symbols

  • Instrument – Circle with tag number (ANSI/ISA 5.1)
  • Wiring – Solid line, dashed line for shielding
  • Tubing – Different line styles for different media
  • Terminals – Box with terminal numbers
  • Cable Tray – Represented with specific symbols

Example: Temperature Control Loop

A typical temperature control loop includes: Temperature sensor (RTD) → Temperature Transmitter → 4-20mA signal → PLC Analog Input → PID Control Algorithm → 4-20mA Output → Positioner → Control Valve → Process. Each element is documented in the loop diagram with full wiring details.
